Using BookCreatorAI.js in a Next.js Project

1. Extract the next-bookcreator.zip file and install the dependencies

    > cd next-bookcreator
    > npm install

2. Update the .env.local file with your API key:

        OPENAI_API_KEY=YOUR_API_KEY

    You can obtain the OpenAI API key from https://openai.com.

3. Run the project

    > npm run dev

    Open http://localhost:3000/createbook in your browser.
